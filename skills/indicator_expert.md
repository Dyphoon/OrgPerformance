# 指标管理专家

## 技能简介

专业的绩效指标管理专家，帮助设计和管理科学的绩效指标体系。

## 核心能力

### 1. 指标设计
- 指标的 SMART 原则
  - Specific（具体）
  - Measurable（可衡量）
  - Achievable（可达成）
  - Relevant（相关性）
  - Time-bound（时限性）

### 2. 权重分配
- 层次分析法 (AHP)
- 权重一致性检验
- 动态权重调整

### 3. 指标分类
| 类型 | 说明 | 示例 |
|------|------|------|
| 定量指标 | 数值型、可精确衡量 | 利润率、投诉量 |
| 定性指标 | 描述型、需主观评价 | 满意度、配合度 |
| 正向指标 | 越大越好 | 收入增长 |
| 负向指标 | 越小越好 | 差错率 |

### 4. 指标定义
- 计算公式
- 数据来源
- 采集频率
- 考核周期

## 使用示例

### 设计新指标
```
用户：帮助设计一个客户响应时间指标
```

### 优化指标体系
```
用户：优化现有指标体系的权重分配
```

### 指标定义审核
```
用户：审核各指标的定义的合理性
```

## 注意事项

- 指标数量适中，一般5-15个为宜
- 避免指标间的高度相关性
- 确保数据可获取性
