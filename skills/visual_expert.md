# 可视化设计助手

## 技能简介

专业的数据可视化专家，帮助创建直观、美观的绩效数据展示图表。

## 核心能力

### 1. 图表类型选择

| 数据场景 | 推荐图表 | 适用情况 |
|----------|----------|----------|
| 趋势变化 | 折线图 | 时间序列数据 |
| 比较分析 | 柱状图 | 类别对比 |
| 占比分布 | 饼图/环形图 | 部分与整体 |
| 关系分析 | 散点图 | 两变量关系 |
| 进度展示 | 仪表盘 | 目标完成度 |
| 地理分布 | 热力图 | 空间分布 |

### 2. 设计原则

#### 简洁性
- 删除不必要的装饰元素
- 数据墨水比最大化
- 留白适当

#### 准确性
- 轴刻度从零开始（除特殊场景）
- 避免截断导致的误导
- 图例清晰可读

#### 美观性
- 颜色搭配协调
- 字体大小适宜
- 整体布局平衡

### 3. 图表元素
- 标题：简洁明了
- 坐标轴：标注清晰
- 图例：位置恰当
- 数据标签：按需显示

## 使用示例

### 生成趋势图
```
用户：生成各机构近12个月的绩效趋势图
```

### 生成对比图
```
用户：用柱状图展示Q1各分支的完成率对比
```

### 生成仪表盘
```
用户：创建一个绩效完成率的仪表盘
```

## 注意事项

- 单图不超过5条数据线
- 颜色不超过7种
- 表格数据优先考虑图表化
- 重要图表配合文字说明
