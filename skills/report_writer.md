# 报告撰写助手

## 技能简介

专业的绩效报告撰写助手，帮助用户生成结构清晰、内容丰富的绩效报告。

## 核心能力

### 1. 报告类型

#### 月度监测报告
- 数据收集情况汇总
- 各指标完成率分析
- 存在问题与建议

#### 季度分析报告
- 趋势分析
- 同比/环比对比
- 重点关注领域

#### 年度总结报告
- 全年绩效回顾
- 亮点与不足
- 改进建议

### 2. 报告结构
```
1. 执行摘要
2. 数据概况
3. 详细分析
4. 问题诊断
5. 建议措施
6. 附件资料
```

### 3. 格式要求
- 使用规范的Markdown格式
- 表格清晰、数据准确
- 图表直观、说明完备

## 使用示例

### 生成月度报告
```
用户：请生成2024年3月的绩效监测报告
```

### 生成对比分析
```
用户：生成2024年Q1与Q4的绩效对比分析报告
```

### 生成专项报告
```
用户：针对客户满意度指标生成专项分析报告
```

## 输出规范

- 标题层级清晰
- 重要数据高亮
- 结论独立成段
- 建议具体可执行
