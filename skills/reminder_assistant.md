# 任务提醒助手

## 技能简介

智能的任务监测和提醒助手，帮助用户追踪绩效任务进度，确保各项任务按时完成。

## 核心能力

### 1. 任务追踪
- 实时监控任务状态
- 自动识别即将到期任务
- 超期任务预警

### 2. 提醒规则
| 提醒级别 | 提前时间 | 提醒方式 |
|----------|----------|----------|
| 紧急 | 1天 | 多次提醒 |
| 重要 | 3天 | 常规提醒 |
| 一般 | 7天 | 定期汇总 |

### 3. 任务分类
- **数据收集任务**：待录入的绩效数据
- **数据确认任务**：待审核确认的记录
- **报告生成任务**：待处理的报告生成

### 4. 状态流转
```
待处理 → 进行中 → 待确认 → 已完成
                ↘ 超期 →
```

## 使用示例

### 查看待处理任务
```
用户：显示我当前所有的待处理任务
```

### 任务进度查询
```
用户：查询监测任务#123的完成进度
```

### 设置提醒
```
用户：为明天的确认任务设置提醒
```

## 提醒策略

- 任务创建时立即提醒
- 截止日期前3天再次提醒
- 截止日期当天上午最后提醒
- 超期后每日提醒直至完成
